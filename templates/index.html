Letterboxd Data Analysis


This website analyzes movies watched by Letterboxd users, showing popular genres, movie release years, directors, actors, actresses, countries, and languages.
The result may take a little time to generate, depending on the number of movies the analyzed user has in their profile, so I ask for some patience while the graphs are being displayed. In the future, I plan to add more analyses and speed up the data extraction process.
If you don't have a Letterboxd account but want to see how it works, you can use this username: bebaagua
Have fun!
