Letterboxd Data Analysis


This website extracts information about the movies that users of the Letterboxd platform consume. Here, you will find out which genres are most watched by any user of the platform, as well as the years of movie releases they have watched the most, the most-watched directors, the most-viewed actors and actresses, the most-watched country and the most-watched language.
The result may take a little time to generate, depending on the number of movies the analyzed user has in their profile, so I ask for some patience while the graphs are being displayed. In the future, I plan to add more analyses and speed up the data extraction process. Please keep in mind that this is a beta version of the project.
If you don't have a Letterboxd account but want to see how it works, you can use this username: bebaagua
Have fun!
